###### Como inserir uma pasta na nuvem(Github) partindo de um arquivo local:

1) Crie um arquivo na pasta em que você iniciou o projeto. 

2)  Arquivo finalizado. Git Bash no repositório em que deseja ex. (main)

3) Digite "git status" e veja que possui um arquivo em destaque vermelho(Isso mostra que o repositório possui mais um arquivo local e não está inserido na nuvem)

4) Digite "Git add ." (Para que esse arquivo vá para "staging area" )

5) Digite  Git commit -m " luz azul"

   (O que há dentro das aspas é um exemplo de descrição que ficará visível para que você reconheça com mais facilidade seu arquivo.) EX. Git commit -m " luz azul"

   6) Sendo todos arquivos commitados 

   7) Digite "git push origin main" ( Levando todos seus arquivos commitados para o GitHub).
